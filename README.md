# Dynamic Task Planner

A responsive and interactive task management web application built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project enables users to add and delete tasks dynamically without refreshing the page while following modern frontend development best practices.

---

## Features

- ✅ Add new tasks dynamically
- ✅ Delete tasks instantly
- ✅ Live task counter
- ✅ Responsive design
- ✅ Semantic HTML structure
- ✅ Accessible interface (Skip Link & ARIA labels)
- ✅ Dynamic DOM manipulation
- ✅ Event delegation for efficient event handling
- ✅ Basic XSS protection using HTML escaping
